Part B — Live Sync

When an import was sourced from Drive, the batch retains a reference to the source file. Editors can trigger a re-sync on any draft import — never on approved outputs (those are frozen).

Part C — PDF Export

Available on the shareable readout page of any approved output. Two destinations.

💾
Download PDF
Generates a PDF of the approved readout and downloads it to your local machine immediately.
📤
Save to Google Drive
Generates the PDF and uploads directly to a chosen Drive folder. Requires Drive auth (same session as import, if connected).

Auth model

Google OAuth scoped to drive.readonly for import/sync and drive.file for export. Token stored per user session — no org-wide service account needed at this stage.
